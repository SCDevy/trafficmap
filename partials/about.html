This application was created as a final project for CSCI 201L at the University of Southern California, taught by Jeffrey Miller. Using the provided JSON traffic data, we have set up a Java server and MySQL database which processes the data and sends it to the Firebase service. The web client can then read and write to Firebase in real-time using the Firebase API, Angular.js, and AngularFire. Graphical representation of data is accomplished using HTML5 canvas and JavaScript.
